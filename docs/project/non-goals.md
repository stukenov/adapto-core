# Non-goals для v1

Не делать в первой версии:

* полноценную замену React;
* universal client-side SPA framework;
* mobile runtime;
* visual builder;
* edge runtime;
* streaming SSR;
* distributed LiveSession cluster;
* full offline mode;
* arbitrary npm compatibility;
* generic public website framework.

Цель v1 -- business app framework, а не универсальная web-платформа.
