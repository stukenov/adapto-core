# Dev server

Dev server должен поддерживать:

* fast rebuild;
* WebSocket hot reload;
* compiler diagnostics;
* route inspection;
* security warnings;
* generated code preview;
* event trace panel;
* state inspector в dev mode.

```bash
adapto dev
```

Output:

```txt
Adapto dev server running at http://localhost:3000
Routes: 14
Components: 42
Live reload: enabled
Security checks: passed
```
